## Run Locally

Clone the project

```bash
  git clone https://github.com/AbidHasanPiash/abidhasanNextjs.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install -y
```

Start the server

```bash
  npm run dev
```
